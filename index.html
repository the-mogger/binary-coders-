About the Project

Our Smart Watering System is a cost-effective solution for plant care. It uses sensors and AI to automate watering and offers live monitoring via a mobile app. Gesture and voice controls make plant management easier than ever!

Features

  • Automatic watering with smart sensors
  • Gesture and voice-controlled operations
  • Live monitoring via mobile app
  • AI-based plant health analysis
  • Cost-effective and easy to install
